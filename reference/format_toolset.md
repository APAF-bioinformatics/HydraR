# Format Toolset for Prompt

Format Toolset for Prompt

## Usage

``` r
format_toolset(tools)
```

## Arguments

- tools:

  List of AgentTool objects.

## Value

A formatted string containing all tool descriptions.

## Examples

``` r
if (FALSE) { # \dontrun{
format_toolset(list(my_tool = AgentTool$new(...)))
} # }
```
