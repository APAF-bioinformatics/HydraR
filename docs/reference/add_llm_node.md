# Create an LLM Agent Node easily

Create an LLM Agent Node easily

## Usage

``` r
add_llm_node(id, role, driver, ...)
```

## Arguments

- id:

  String. Unique identifier for the node.

- role:

  String. System prompt/role for the agent.

- driver:

  AgentDriver object.

- ...:

  Additional arguments passed to AgentLLMNode\$new()

## Value

AgentLLMNode object.
